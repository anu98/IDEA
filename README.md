## Overview

IDEA is an immersive augmented reality (AR) application designed to support the exploration and analysis of multidimensional and volumetric datasets through natural interaction paradigms.

Developed as part of my Master's thesis, the project introduces an intuitive spatial user interface that replaces traditional desktop workflows with an interactive virtual bookshelf. Datasets are organized as virtual books, allowing users to browse, inspect, and visualize scientific data directly within their physical environment using a head-mounted display.

The project focuses on improving usability and user experience for immersive data exploration while leveraging existing visualization techniques in an AR environment.

Important note : This repository builds upon the original AugmeNDT framework developed by Alexander Gall and collaborators.

This version was used and extended as part of a Master's thesis project focusing on immersive UI/UX design and interaction paradigms for AR-based data exploration.

Modifications primarily focus on:
- AR user interface and interaction design
- Dataset browsing and navigation workflows
- Integration and presentation of visualization modules in immersive environments
- UX improvements for spatial data exploration

Original system architecture and core visualization methods are attributed to the original authors (see citation section below).

## Key Features

### 📚 Virtual Bookshelf Interface
- Organizes datasets as virtual books
- Represents folder hierarchies using a bookshelf metaphor
- Enables intuitive spatial navigation

### 📖 Interactive Book Navigation
- Browse datasets through realistic page flipping
- Dataset previews displayed directly on pages
- Metadata presented before loading visualizations

### 📊 Scientific Data Visualization
Supports multiple visualization techniques, including:

- Scatter Plots
- Bar Charts
- Multidimensional Distribution (MDD) Glyphs
- CT Volume Rendering

### 🎨 Visualization Customization
Users can modify visualization properties within AR, including:

- Color adjustments
- Display settings
- Visualization parameters

### 🥽 Natural Interaction
Designed specifically for head-mounted AR devices using:

- Hand interactions
- Spatial UI
- Direct manipulation
- Immersive 3D interfaces

## System Workflow

1. Browse datasets using the virtual bookshelf.
2. Open a virtual book representing a dataset category.
3. Flip through pages containing dataset previews and metadata.
4. Select a dataset.
5. Visualize the data in immersive 3D.
6. Adjust visualization properties through the AR interface.
